# DigiJet

Site vitrine de DigiJet, une activité de création de sites internet orientée visibilité locale, image professionnelle et solutions web accessibles.
https://ryanjpm.github.io/digijet/

## Présentation

Ce projet est un site vitrine développé en PHP, HTML, CSS et JavaScript.  
Il présente les services DigiJet, les tarifs, une page de contact, les mentions légales et la politique de confidentialité.

## Technologies utilisées

- PHP
- HTML5
- CSS3
- JavaScript
- PHPMailer
- Composer
